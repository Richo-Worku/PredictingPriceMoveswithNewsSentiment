# Financial News Sentiment Analysis

## Project Overview
This project analyzes financial news headlines and market data to identify sentiment trends and potential impacts on stock price movements.

The project includes:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Sentiment analysis
- Technical indicators
- Visualization dashboards

---




### Clone the repository

```bash
git clone https://github.com/Richo-Worku/PredictingPriceMoveswithNewsSentiment
```

### Create virtual environment

```bash
python -m venv venv
```

### Activate virtual environment

#### Windows
```bash
venv\Scripts\activate


### Install dependencies

```bash
pip install -r requirements.txt
```

## Running the Project
